# Corpus Linguistics -- WiSe 2025-2026
Content: Docker files for jupyter servers, corpora, scripts

This repository contains teaching materials from current and previous classes at the University of Stuttgart.

## Upcoming
 - Scripts for retrieving phenomena of language acquisition in CHILDES corpora (German/English/French)

## Corpora
 - Korpus Lehramt Französisch: novels, theater pieces, poetry, film subtitles... from sources selected by the students in order to reflect the state of French they studied
 - German corpus of science fiction: earliest science fiction texts in German found on Wikisource

They are used to create shared project on the platform "Jupyter für die Lehre".
